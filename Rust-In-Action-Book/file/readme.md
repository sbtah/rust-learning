Generalnie to jest HnS wiec nie wiem skad zdziwienie ze ludzie chca wszystko rozpierdalac, problem progresu na wczesnych poziomach jest ogromny, tym bardziej ze wyglada na to ze znowu znerfili drop orbow bo ja do 3 aktu 1 exa i jednego alchemy orba wydropilem, tu nawet nie ma z czego ulepic dmg na wczesnych etapach i tak uzywam combo skilli tylko jesli kogos bawi rzucanie 3 roznych skill po kilka razy zeby na koniec wszystko wybuchlo i zadalo srednio duzy dmg to spoko, mnie to jakos nie bawi, z reszta samo GGG sie do tego poniekad przyznalo obcinajac zycie potworom na wczesnych etapach o 25%.

Tu nawet pro gracze narzekaja nie zadni casuale.
